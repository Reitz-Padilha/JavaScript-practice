# Ex01 - Interação com usuário

## Descrição
Primeiro exercício de JavaScript com interação usando alert, confirm e prompt.

## Recursos utilizados
- window.alert()
- window.confirm()
- window.prompt()

## O que aprendi
- Como exibir mensagens na tela
- Como interagir com o usuário
- Uso básico de JavaScript no HTML
